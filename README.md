# ModernShop - E-Commerce Website

## 📌 Project Overview
ModernShop is a simple front-end E-Commerce website developed using HTML, CSS, and JavaScript. It simulates an online shopping system where users can browse products, view product details, add products to a cart, register/login, and place orders. User data, cart information, and order history are stored using Local Storage and Session Storage.

---

## 🚀 Features

### 🛍 Product Management
- Display all products dynamically.
- Product details popup (Modal).
- Category-based product listing.
- Responsive product cards.

### 👤 User Authentication
- User Registration.
- User Login.
- Logout functionality.
- Session management using Session Storage.

### 🛒 Shopping Cart
- Add products to cart.
- Increase or decrease quantity.
- Remove products from cart.
- Cart item count badge.
- Automatic total calculation with tax.

### 📦 Order Management
- Checkout process.
- Generate Order ID automatically.
- Save order history.
- Display previous orders.

### 🔔 Notifications
- Toast notifications for:
  - Successful login
  - Registration
  - Add to cart
  - Checkout
  - Errors

### 💾 Storage
- Local Storage for:
  - Users
  - Cart data
  - Order history

- Session Storage for:
  - Current logged-in user
